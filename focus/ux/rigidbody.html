Change Shape

Changes the physics bounding box for the object.

Also see: Blender Manual

Copy from Active

Copies physics settings from the active object and applies it to all other selected objects.

Also see: Blender Manual

Add RB Passive

Adds passive rigidbody physics to the selected objects.

Also see: Blender Manual

Connect Constraints

Adds rigidbody constraints between selected objects.

Also see: Blender Manual

Remove RB

Removes a rigidbody modifier on the selected object.

Also see: Blender Manual

Bake to Keyframes

Bakes the simulated physics action to transformation keyframes.

Also see: Blender Manual

Add RB Active

Adds active rigidbody physics to the selected objects.

Also see: Blender Manual

Calculate Mass

Calculates mass values for the object based on volume.

Also see: Blender Manual
