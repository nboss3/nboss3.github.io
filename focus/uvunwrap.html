Smart Project

Functions exactly the same as the default Smart UV Project function in Blender.

Also see: Blender Manual

Unwrap All Menu

Unwraps all objects in the scene using the specified unwrap method.

Pack Unwrap All

Unwraps all objects using the Pack Unwrap method.

Sharp Unwrap All

Unwraps all objects using the Sharp Unwrap method.

Cube Pack All

Unwraps all objects using the Cube Pack method.

Sphere Pack All

Unwraps all objects using the Sphere Pack method.

Cylinder Pack All

Unwraps all objects using the Cylinder Pack method.

Unwrap by Texture Set Menu

Unwraps visible mesh objects in groups based on their material. Objects with the same material will be unwrapped onto the same UV map. To exclude objects from this process, simply hide them.

Pack Unwrap by Texture Set

Unwraps by texture set with the Pack Unwrap method.

Sharp Unwrap by Texture Set

Unwraps by texture set with the Sharp Unwrap method.

Cube Unwrap by Texture Set

Unwraps by texture set with the Cube Pack method.

Sphere Unwrap by Texture Set

Unwraps by texture set with the Sphere Pack method.

Cylinder Unwrap by Texture Set

Unwraps by texture set with the Cylinder Pack method.

Unwrap Sharp

Marks UV seams on sharp edges of object. Unwraps the object and packs UV islands

UV Project Pack Menu

Unwraps objects using the specified UV projection method and packs UV islands.

UV Projects with the cube projection method then packs UV islands.

Also see: Blender Manual

Sphere Pack Projection

UV Projects with the sphere projection method then packs UV islands.

Also see: Blender Manual

Cylinder Pack Projection

UV Projects with the cylinder projection method then packs UV islands.

Also see: Blender Manual
