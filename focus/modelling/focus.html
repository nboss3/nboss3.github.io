Object Mode

Activates a mode which allows rapid object adding for blocking out a model. Adding objects can be done by just pressing the spacebar rather than having to navigate menus. All editing functions are available in this mode except one that require the hotkeys listed in the overlay menu.

Boolean Mode (ND)

Enters an ND (non-destructive) boolean mode. All keymaps are shown in the 3D View overlay. The mode dynamically adjusts modifiers on the object as you create boolean operations.

Kitbash Menu

Opens the kitbash menu; an interactive browser for quickly adding in fbx files to the scene. For best results, place all your fbx files you want to kitbash with in one folder and define it in the user preferences.

Render Previews

To use the Render Previews function, you must have a camera and one material in your blender scene. You first select a location where your FBX files are located, then the function renders a jpg image at a resolution of 256x256 for each FBX in the selected folder. The previews then automatically show in the browser.
